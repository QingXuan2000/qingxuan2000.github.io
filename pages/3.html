1.3 文本样式¶

  • 粗体文本 - 使用 **粗体** 或 __粗体__
  • 斜体文本 - 使用 *斜体* 或 _斜体_
  • 粗斜体 - 使用 ***粗斜体***
  • ~~删除线~~ - 使用 ~~删除线~~ (如果支持)
  • 行内代码 - 使用反引号
  • 下划线 - 使用 HTML <u> 标签
